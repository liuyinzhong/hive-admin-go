## 项目基本说明

1. 使用 go 语言开发
2. 数据库使用 MySql
3. 数据库账号:root，密码:123456。端口:3306
4. 项目接口使用本地 9191 端口启动
5. 数据库表全局都采用UUID(带横线)格式，示例：cc1a8564-37e7-47df-ad60-6c0a7f199d31
6. 时间格式，全局采用 YYYY-MM-DD HH:mm:ss 格式；示例：2026-05-18 15:30:26
7. 时区规范，全局采用 time.ParseInLocation (本地时区)
7. 用户鉴权信息，在接口请求头里 Authorization(有Bearer 前缀)。格式为：Bearer xxxxx
9. 响应到接口的字段。统一小驼峰写法
10. 排序参数规范：sorts=avatar,desc;username,desc （query参数）
11. 多选查询参数规范：storyStatus=1,2,3&name=

## 接口命名规范举例

| 名称              | 请求地址                          | 请求方式 | 参数说明                                                     |
| ----------------- | --------------------------------- | -------- | ------------------------------------------------------------ |
| 用户列表 (分页)   | /api/system/users                 | GET      | Query：page、pageSize + 业务筛选参数 + 排序参数?<br />举例：page=1&pageSize=20&sorts=avatar,desc;username,desc&username=&realName=&status=1,2 |
| 用户列表 (不分页) | /api/system/users/all             | GET      | 可携带业务筛选参数                                           |
| 创建用户          | /api/system/users                 | POST     | JSON Body 请求体                                             |
| 删除用户          | /api/system/users                 | DELETE   | JSON Body：ID 数组 `[1,2,3]`                                 |
| 用户详情          | /api/system/users/{userId}        | GET      | Path 路径参数：userId                                        |
| 修改用户          | /api/system/users/{userId}        | PUT      | Path 路径参数 + JSON Body 请求体                             |
| 修改用户状态      | /api/system/users/{userId}/status | PUT      | Path 路径参数 + JSON Body 状态参数                           |
| 导出用户          | /api/system/users/export          | GET      | Query 参数与分页列表接口保持一致                             |
| 导入用户          | /api/system/users/import          | POST     | 表单文件上传 file                                            |

## 响应格式

```json
{
  "code": 0, // 业务状态码（核心） 成功时为0，失败时为 -1
  "data": null,// 成功时的业务数据 对象或数组
  "error": null,// 失败时的错误详情（对象/结构化信息）
  "message": "ok"// 给用户/开发者看的提示文本
}
```

## 分页入参格式(GET请求格式)

```json
{
  "page": 0, //页码
  "pageSize": 20, //页数量
  "sorts":"avatar,desc;username,desc"
}
```

## 分页响应格式

```json
{
  "code": 0,
  "data": {
	items:[], //根据分页参数查询出来的数据集合
	total:0, //数据库里查出来的总数量
	},
  "error": null,
  "message": "ok"
}
```
